<a name="how-do-i-create-a-custom-provider-for-logging-health-monitoring-events"></a>[Jak i.] Tworzenie niestandardowego dostawcy dla rejestrowania kondycji monitorowanie zdarzeń
====================
przez [Pels Krzysztof](https://twitter.com/chrispels)

Krzysztof Pels przedstawiono sposób tworzenia niestandardowego dostawcy dla kondycji ASP.NET monitorowanie systemu, który rejestruje zdarzenia w pliku tekstowym dziennika monitorowania kondycji. Dowiedz się, jak utworzyć niestandardowego dostawcę klasy, która dziedziczy po klasie BufferedWebEventProvider, zrozumieć metody i właściwości służące do tworzenia dostawcy i utworzyć witrynę sieci web, która używa go do zapisania plików dziennika.
