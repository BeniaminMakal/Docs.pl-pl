<a name="how-do-i-extend-and-customize-an-aspnet-server-control-for-a-specific-purpose"></a>[Jak i.] Rozszerzanie i dostosowywanie kontrolka serwera programu ASP.NET w określonym celu
====================
przez [Pels Krzysztof](https://twitter.com/chrispels)

W tym wideo Pels Krzysztof będzie pokazują, jak rozszerzyć formantu standardowego serwera ASP.NET i dostosować go do określonego celu. Specjalne elementy sterujące udostępniają wygodny sposób Implementowanie elementów interfejsu użytkownika z znormalizowanych wielu witryn sieci web dla konkretnych osób lub zespołów deweloperów. W tym przykładzie Zobacz jak rozszerzyć kontroli DropDownList utworzyć kontrolkę wyboru specjalne roku. Dowiedz się, jak dodać właściwości dla atrybutów niestandardowych, które określają zachowanie zakresu lat, które mogą być wyświetlane. Następnie zobacz, jak te atrybuty niestandardowe można ustawić w składni deklaratywnej, takich jak standardowe atrybuty kontrolki. Zobacz, jak kilka dodatkowych właściwości można następnie dodać do oferowanie dodatkowych funkcji kontroli zachowania listy. Na koniec Zobacz, jak formantu rozszerzonego serwera ASP.NET mogą zostać przeniesione do osobny zestaw, dzięki czemu można używać wielu witrynach sieci web.
