przez [Windsor Tomasz](https://twitter.com/robwindsor)

Gdy wystąpi błąd podczas przetwarzania odświeżania strony wyzwalane przez element UpdatePanel ASP.NET AJAX, elementu PageRequestManager po stronie klienta wyświetli alert (.NET w wersji 2.0 lub 3.0) lub po prostu ponownie zgłosić wyjątek (.NET 3.5 lub nowszej).
