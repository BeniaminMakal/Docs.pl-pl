<a name="how-do-i-share-data-between-client-javascript-and-server-code-using-a-hidden-field"></a>[Jak mogę] Udostępnianie danych między kodem JavaScript klienta i kodem serwera za pomocą ukrytego pola
====================
przez [Chris pikseli](https://twitter.com/chrispels)

W tym wideo pikseli Chris pokazują, jak udostępniać dane między JavaScript i kodu po stronie serwera za pomocą ukrytego pola na stronie ASP.NET. Dowiedz się, jak programowo utworzyć ukryte pole na kliencie przy użyciu metody RegisterHiddenField w kodzie po stronie serwera. Ponadto zobacz, jak programowo utworzyć formant serwera formant HiddenField na potrzeby udostępniania danych na stronie ASP.NET. Na koniec użyj metody RegisterArrayDeclaration Aby utworzyć kontener w kodzie serwera do przechowywania wielu wartości, który jest renderowany do strony gdzie on dostępny w dla języka JavaScript.
