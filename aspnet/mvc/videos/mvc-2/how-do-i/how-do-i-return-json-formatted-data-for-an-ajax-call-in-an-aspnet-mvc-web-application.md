<a name="how-do-i-return-json-formatted-data-for-an-ajax-call-in-an-aspnet-mvc-web-application"></a>Jak I: zwracać w formacie JSON dane dla wywołania AJAX w aplikacji sieci Web platformy ASP.NET MVC?
====================
przez [Pels Krzysztof](https://twitter.com/chrispels)

W tym wideo Pels Krzysztof przedstawia sposób użycia klasy JsonResult i Json do zwrócenia wystąpienia klas w formacie JSON format danych. Najpierw przykładowej aplikacji sieci web MVC jest tworzony i kontrolera i widoku skojarzonego są dodawane do informacji o kliencie. Klasa odbiorcy proste tworzona jest częścią modelu danych. Funkcja GetCustomer() jest dodawane do kontrolera klienta do pobierania danych dla jednego klienta w formacie JSON. Na koniec skrypt zostanie dodany do widoku klienta, który używa jQuery do wywoływania AJAX GetCustomer() można pobrać danych klienta w formacie JSON
