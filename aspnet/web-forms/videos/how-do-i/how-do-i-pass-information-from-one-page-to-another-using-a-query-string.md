<a name="how-do-i-pass-information-from-one-page-to-another-using-a-query-string"></a>[Jak i.] Przekazywania informacji z jednej strony przy użyciu ciągu zapytania
====================
przez [Pels Krzysztof](https://twitter.com/chrispels)

To wideo Pels Krzysztof opisano sposób informacje mogą zostać przekazane z jednej strony przy użyciu ciągu zapytania. Najpierw zobacz sposób tworzenia ciągu zapytania w tym nazwy zmiennych i wartości z informacji na temat formularzem wprowadzania danych. Następnie zobacz, jak uzyskać dostęp do informacji o ciągu zapytania, gdy strona jest przesyłana z tym, jak wiele zmiennych ciągu zapytania są obsługiwane. W ramach przetwarzania po stronie serwera informacji o ciągu zapytania zobacz temat jak zaimplementować weryfikację istnienia zmiennych ciągu zapytania i jeśli zawierają one oczekiwanego typu danych. Na koniec Dowiedz się, jak przekazać w tym spacji i znaku w ciągu zapytania przez kodowanie ciągu znaków specjalnych.
