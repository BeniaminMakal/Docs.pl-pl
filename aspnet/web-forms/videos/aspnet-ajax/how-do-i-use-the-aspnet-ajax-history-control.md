<a name="how-do-i-use-the-aspnet-ajax-history-control"></a>[Jak i.] Formant ASP.NET AJAX historii?
====================
przez [Scott Golightly](https://github.com/scottgolightly)

Jednym z problemów napotykają Typowa aplikacja AJAX jest aktualizacja częściowa strona nie została dodana do historii przeglądarki sieci Web. Oznacza to, że nie cofnąć krok AJAX przycisku Wstecz w przeglądarce, ale przeniesienie jednej całego dokumentu, który jest mało prawdopodobne, aby użytkownik oczekuje. Kontrolki ASP.NET AJAX historii umożliwia deweloperowi Wstaw punkty historii, dzięki czemu użytkownik może kliknąć w przeglądarce przycisków Wstecz i dalej można przenieść między krokami AJAX.
