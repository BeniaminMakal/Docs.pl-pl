<a name="measuring-the-business-value-of-ajax"></a>Pomiaru wartość biznesowa AJAX
====================
przez Lewandowski marek

ASP.NET AJAX jest dobrze znany dla możliwość ulepszyć środowisko użytkownika w aplikacji sieci Web. Przy użyciu programu Visual Studio Team System można również mierzyć wartościach biznesowych dodawania ASP.NET AJAX do aplikacji sieci Web. W szczególności można użyć funkcji badania sieci Web i obciążenia programu Visual Studio Team System do eksplorowania jak różnica w rozmiarze transmisji danych HTTP wpływa na wydajność i skalowalność właściwości aplikacji sieci Web.
