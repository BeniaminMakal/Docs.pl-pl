<a name="older-versions---ajax-control-toolkit"></a>Starsze wersje - Toolkit kontroli Ajax
====================
> Jak rozpocząć pracę z zestawu narzędzi kontroli Ajax i jak korzystać z niektórych formantów najpopularniejszych. DevExpress utrzymuje bezpłatne i otwórz powierzając jej ich konserwację Toolkit kontroli Ajax.
> 
> 
> ![](index/_static/image1.png)DevExpress, deweloperzy wielokrotnie nagradzany formularzy sieci Web ASP.NET, MVC i formantów interfejsu użytkownika HTML5 i struktur, przejął konserwacji zestawu ASP.NET AJAX kontroli narzędzi, aby przywrócić tę bibliotekę open source wysokiej wymagane do stanu niezawodny. [Dowiedz się więcej](https://go.devexpress.com/AjaxControlToolkit_ASP_Resources_Tutorials.aspx).
