<a name="how-do-i-determine-whether-to-use-a-web-site-or-a-web-application-project"></a>[Jak i.] Określanie, czy używać witryny sieci Web lub projektu aplikacji sieci Web
====================
przez [Pels Krzysztof](https://twitter.com/chrispels)

W tym wideo Pels Krzysztof będzie natomiast różnice między projekt witryny sieci web ASP.NET i projektu aplikacji sieci web. Dowiedz się zagadnienia dotyczące korzystania z jednego typu projektu lub innych. Po pierwsze informacje o głównych atrybuty projektu aplikacji sieci web, łącznie z jej każdego folderu. Następnie zobacz atrybuty głównych projekt witryny sieci web i jej tradycyjnych charakteru na podstawie projektu programu Visual Studio, co zapewnia więcej szczegółowości i elastyczność, takie jak dodawanie klasy, dotyczących przestrzeni nazw i odwołania do kontrolki użytkownika lub strony programowo. Następnie omówiono modelu kompilacji dla każdego typu witryny sieci web oraz rozwiązań alternatywnych, obejmujących użycie aspnet\_compiler.exe i aspnet\_merge.exe narzędzia i wdrażania projektu sieci web programu Visual Studio 2008.
