## <a name="conventions"></a><span data-ttu-id="8966d-330">Konwencje</span><span class="sxs-lookup"><span data-stu-id="8966d-330">Conventions</span></span>

<span data-ttu-id="8966d-331">Ilość kodu musiały zapisu w kolejności Entity Framework można było utworzyć pełnej bazy danych dla Ciebie jest minimalny, z powodu użycia *konwencje*, lub założenia, które wprowadza programu Entity Framework.</span><span class="sxs-lookup"><span data-stu-id="8966d-331">The amount of code you had to write in order for the Entity Framework to be able to create a complete database for you is minimal because of the use of *conventions*, or assumptions that the Entity Framework makes.</span></span> <span data-ttu-id="8966d-332">Niektóre z nich ma już zauważyć lub użyto bez Twojej wiedziały z nich:</span><span class="sxs-lookup"><span data-stu-id="8966d-332">Some of them have already been noted or were used without your being aware of them:</span></span>

- <span data-ttu-id="8966d-333">Pluralized formy nazwy klas jednostki są używane jako nazwy tabeli.</span><span class="sxs-lookup"><span data-stu-id="8966d-333">The pluralized forms of entity class names are used as table names.</span></span>
- <span data-ttu-id="8966d-334">Nazwy właściwości jednostki są używane dla nazw kolumn.</span><span class="sxs-lookup"><span data-stu-id="8966d-334">Entity property names are used for column names.</span></span>
- <span data-ttu-id="8966d-335">Właściwości jednostki, które są nazywane `ID` lub *classname* `ID` są rozpoznawane jako właściwości klucza podstawowego.</span><span class="sxs-lookup"><span data-stu-id="8966d-335">Entity properties that are named `ID` or *classname* `ID` are recognized as primary key properties.</span></span>
- <span data-ttu-id="8966d-336">Właściwość jest interpretowana jako właściwości klucza obcego, jeśli jest o nazwie  *&lt;nazwą właściwości nawigacji&gt;&lt;nazwa właściwości klucza podstawowego&gt;*  (na przykład `StudentID` dla `Student` właściwość nawigacji, ponieważ `Student` klucza podstawowego jednostki jest `ID`).</span><span class="sxs-lookup"><span data-stu-id="8966d-336">A property is interpreted as a foreign key property if it's named *&lt;navigation property name&gt;&lt;primary key property name&gt;* (for example, `StudentID` for the `Student` navigation property since the `Student` entity's primary key is `ID`).</span></span> <span data-ttu-id="8966d-337">Właściwości klucza obcego może również być taką samą nazwę po prostu &lt;nazwa właściwości klucza podstawowego&gt; (na przykład `EnrollmentID` ponieważ `Enrollment` klucza podstawowego jednostki jest `EnrollmentID`).</span><span class="sxs-lookup"><span data-stu-id="8966d-337">Foreign key properties can also be named the same simply &lt;primary key property name&gt; (for example, `EnrollmentID` since the `Enrollment` entity's primary key is `EnrollmentID`).</span></span>
