<a name="how-do-i-use-roles-to-segment-functionality-in-an-aspnet-web-site"></a>[Jak i.] Użyć ról na segmenty funkcji w witrynie sieci Web ASP.NET
====================
przez [Pels Krzysztof](https://twitter.com/chrispels)

To wideo Pels Krzysztof wyświetli jak zwiększyć użycie ról w witrynie sieci web ASP.NET po skonfigurowaniu członkostwa ASP.NET. Po pierwsze Dowiedz się, jak można skonfigurować za pomocą menedżera ról wbudowanych w programie Visual Studio .NET ról. Zobacz sposób definiowania ról w zależności od potrzeb biznesowych skojarzone z witryną sieci web. Po zatwierdzeniu role instalacji jak struktury witryny sieci web z punktu widzenia hierarchii folderów i strony do najbardziej wydajnie ról i potrzeb biznesowych. W ramach tego zobacz sposób użycia atrybutu ról podczas definiowania autoryzacji w pliku web.config, skojarzone z każdego folderu. Sprawdź też, jak używać &lt;lokalizacji&gt; element, aby zastosować ustawienia autoryzacji do określonej lokalizacji, takich jak strony.
