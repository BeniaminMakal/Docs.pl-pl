<a name="how-do-i-use-the-conditional-updatemode-of-the-updatepanel"></a>[Jak i.] Użyj warunkowego UpdateMode elementu UpdatePanel?
====================
przez [Stagner Jan](https://github.com/JoeStagner)

Element UpdatePanel ASP.NET AJAX zawiera właściwość UpdateMode, która może być ustawiony na "Zawsze" lub "Warunkowy". Wartość domyślna to zawsze, w którym to przypadku UpdatePanel zawsze zaktualizuje jego zawartości podczas ogłaszania zwrotnego asychronous. W tym wideo możemy Dowiedz się, jak możemy ustawić właściwość UpdateMode Conditional, w których przypadku UpdatePanel tylko zaktualizuje jego zawartości po jego metody aktualizacji naszego kodu po stronie serwera. Dzięki temu można używać logikę warunkową w języku C# lub kod Visual Basic w celu określenia, czy element UpdatePanel zaktualizuje jego zawartości podczas asynchronicznego ogłaszania zwrotnego.
