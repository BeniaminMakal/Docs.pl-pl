# <a name="community-oss-authentication-options"></a>Opcje uwierzytelniania społeczności OSS

Ta strona zawiera opcje uwierzytelniania dostarczane przez społeczność typu open source dla platformy ASP.NET Core. Ta strona będzie okresowo aktualizowana jako nowego dostawcy staną się dostępne.

# <a name="oss-authentication-providers"></a>Dostawców uwierzytelniania OSS

Poniższa lista jest sortowana alfabetycznie.

| Nazwa | Opis |
|:--------------|:------------------|
| [AspNet.Security.OpenIdConnect.Server (ASOS)](https://github.com/aspnet-contrib/AspNet.Security.OpenIdConnect.Server) | Min.-poziomu/protokołu pierwszego OpenID Connect serwera framework dla platformy ASP.NET Core i OWIN/Katana |
| [IdentityServer4](https://identityserver.io/) | Framework OpenID Connect i OAuth 2.0 dla platformy ASP.NET Core - zakwalifikowane OpenID Foundation, a w obszarze Zarządzanie .NET Foundation |
| [OpenIddict](https://github.com/openiddict/openiddict-core) | Serwer protokołu OpenID Connect łatwy w użyciu dla platformy ASP.NET Core  |
| [Cierge](https://github.com/pwdless/Cierge) | Passwordless, dzięki wsuwanej konstrukcji uwierzytelniania OpenID Connect   |

Można uzyskać dostawcy tu dodać [edytować tę stronę](https://github.com/login?return_to=https%3A%2F%2Fgithub.com%2Faspnet%2FDocs%2Fedit%2Fmaster%2Faspnetcore%2Fsecurity%2Fauthentication%2Fcommunity.md).
