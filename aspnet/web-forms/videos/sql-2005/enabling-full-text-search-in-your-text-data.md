<a name="enabling-full-text-search-in-your-text-data"></a>Włączanie wyszukiwania pełnotekstowego w Twoich danych tekstu
====================
przez [firmy Microsoft](https://github.com/microsoft)

Wyszukiwanie pełnotekstowe umożliwia zapisywanie dużych fragmentów tekstu — nawet dokumentów Microsoft Word lub innych formatów plików — do bazy danych i wykonywać złożonych zapytań na podstawie tego tekstową danych a oczekiwano doskonałą wydajność. Tej lekcji pokazano, jak skonfigurować wyszukiwanie pełnotekstowe, zwracając szczególną uwagę na wszystkie szczegóły konfiguracji w programie SQL Server Express. Następnie możemy przenieść do tworzenia indeksów pełnotekstowych, konfigurowanie ustawień wypełniania indeksu i wykonywanie zapytań, które pokazują predykatu CONTAINS i jego wiele, wiele zmian.
