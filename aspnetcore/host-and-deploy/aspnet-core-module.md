<span data-ttu-id="681f0-109">Następujące *web.config* pliku została opublikowana na potrzeby [wdrożenia zależne od framework](/dotnet/articles/core/deploying/#framework-dependent-deployments-fdd) i konfiguruje moduł Core ASP.NET do obsługi żądań lokacji:</span><span class="sxs-lookup"><span data-stu-id="681f0-109">The following *web.config* file is published for a [framework-dependent deployment](/dotnet/articles/core/deploying/#framework-dependent-deployments-fdd) and configures the ASP.NET Core Module to handle site requests:</span></span>

```xml
<?xml version="1.0" encoding="utf-8"?>

  <system.webServer>
    <handlers>
      <add name="aspNetCore" path="*" verb="*" modules="AspNetCoreModule" resourceType="Unspecified" />
    </handlers>
    <aspNetCore processPath="dotnet" 
                arguments=".\MyApp.dll" 
                stdoutLogEnabled="false" 
                stdoutLogFile=".\logs\stdout" />
  </system.webServer>
</configuration>
```
