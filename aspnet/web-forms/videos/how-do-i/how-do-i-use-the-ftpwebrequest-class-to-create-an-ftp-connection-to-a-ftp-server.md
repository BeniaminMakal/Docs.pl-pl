<a name="how-do-i-use-the-ftpwebrequest-class-to-create-an-ftp-connection-to-a-ftp-server"></a>[Jak i.] Użyj klasy FtpWebRequest, aby utworzyć połączenie FTP do serwera FTP
====================
przez [Pels Krzysztof](https://twitter.com/chrispels)

W tym wideo Pels Krzysztof przedstawia sposób użycia klasy FtpWebRequest, aby utworzyć połączenie FTP i transferu pliku. Najpierw należy utworzyć prosty .aspx formularza w celu wybrania pliku, aby przesłać do serwera FTP. Następnie możesz zapoznać się niezbędne metody i właściwości klasy FtpWebRequest, które trzeba skonfigurować, aby ustanowić połączenie FTP. Następnie zobacz temat jak poznasz zawartość pliku lokalnego, dzięki czemu plik można przenieść za pomocą StreamReader i element StreamWriter zapisać go do FtpWebRequest. Na koniec FtpWebResponse są uzyskiwane Aby określić wyniki transferu. Takie podejście może służyć do programowego transferu plików dla aplikacji sieci web.
